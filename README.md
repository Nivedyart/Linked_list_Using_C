# Linked list Using C programming language

**Linked list that stores data of type int(using struct) with Properties :**

- Function for adding a new node
- Function for deleting a particular node (referenced by the location)
- Function to delete all the nodes from the list which contain a particular data say a number 5
- Function to delete the complete linked list
- Function to display the linked list
- Function to display the inverted linked list
- Function to display the total memory space occupied by the linked list
- Function to delete all the nodes from the list which contain a particular data say a number 5 and the next subsequent node
